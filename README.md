# 프로세카 레이팅 프로젝트 입니다.

- 목표
1. 곡 정보의 DB화
2. 전체 곡의 보면상수 정립
3. 보면상수를 와 판정 기반으로 (가칭)테크니컬 스코어(Technical Score/T.S)산출
4. T.S를 기반으로 자신의 실력을 판별할 수 있는 점수 산출

- 단계 / 계획
1. 곡정보 DB화 (이름, 번역명, 작곡가, 난이도별 레벨, 업데이트 날짜 등)
2. 보면상수 기초 정립 (-> 상시로 수정 및 업데이트시 추가 및 갱신)
3. T.S 산출 수식 정립
4. 프로그램 제작 (python 이용예정)
5. 웹용 프로그램으로 컨버트
6. 웹 서버로 운용 or 파일 배포
